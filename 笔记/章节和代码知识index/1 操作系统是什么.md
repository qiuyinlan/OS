## 问题 5：启动流程总结

### 从上电到 Loader，整个流程是怎样的？

[[BIOS]]
[[MBR主引导记录]]
[[Loader加载器]]

```
上电 → BIOS 初始化硬件 → BIOS 加载 MBR (0x7C00)
    → MBR 加载 Loader → Loader 加载内核 → 操作系统运行
```
