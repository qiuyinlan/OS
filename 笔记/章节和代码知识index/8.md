# **🌟 第四个问题：一个进程的“堆”和“栈”是怎么在内存中管理的？**

### 1. **栈（Stack）**：自动增长、自动释放

- 放局部变量、函数参数、返回地址
    
- 地址向 **低地址**增长
    
- 内核只需要给栈预留空间，不需要动态扩展
    
- 每次 `call` / `ret` 让栈自然变化
    

### 2. **堆（Heap）**：程序主动申请

- 用 `malloc/free` 或 `new/delete`
    
- 地址向 **高地址**增长
    
- 由内核或用户态分配器通过**分页 + 位图**组合管理
    
- 本质上就是：程序要更多内存 → 申请更多“虚拟页”
