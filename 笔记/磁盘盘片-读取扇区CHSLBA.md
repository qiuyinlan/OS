# 磁道与扇区
磁道（**Track**）其实是——  
👉 磁盘表面上一条**狭窄的环形磁带路径**。  
它的宽度非常小，大约只有几微米（μm）级别。

磁盘表面是涂有**磁性材料**的（比如氧化铁粉），  
磁头能在上面**改变这些微小区域的磁化方向**，  
就像“写入0和1”的信号。


# 扇区

## 扇区编号

- 扇区在每条 **磁道** 上从 0 开始编号。
    
- **磁道**可以理解为盘片上的同心圆。
    
- 不同磁道的扇区编号可以重复，比如磁道 0 的扇区 0，磁道 1 也有扇区 0。
    

## 扇区头（Sector Header）

- 每个扇区前面有==一点点信息，叫 **扇区头**==。
    
- 它告诉硬盘：
    
    1. 当前磁头号（Head）——用哪块盘片？
        
    2. 当前磁道号（Cylinder）——用哪条同心圆？
        
    3. 当前扇区号（Sector）——这一条磁道上的哪块？
        
- 磁头就是靠这个信息找到你想读或写的数据。

# 读取扇区的方式

### chs
柱面，磁头，扇区
cylinder,heads,sector
 

### LBA（Logical Block Addressing）

[[CHSLBA]]

>逻辑块寻址

lba28
最大寻址范围为28位，128gb

lba48,256tb
它的核心思想非常简单：

> 不再用 “柱面/磁头/扇区” 三个参数表示位置，  
> 而是—— **给所有扇区编上一个从 0 开始的编号**。

所以每个扇区都有一个 **唯一编号**（就像身份证号），  
BIOS、操作系统、磁盘控制器都可以直接按编号访问。
