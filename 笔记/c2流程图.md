MBR 执行流程图

  ┌─────────────────────────────────────┐
  │   BIOS 加载 MBR 到 0x7c00           │
  │   设置 cs=0, ip=0x7c00              │
  └─────────────┬───────────────────────┘
                ↓
  ┌─────────────────────────────────────┐
  │ 1. 初始化段寄存器                    │
  │    mov ax, cs                       │
  │    mov ds/es/ss/fs, ax (全部=0)     │
  │    mov sp, 0x7c00 (栈顶)            │
  └─────────────┬───────────────────────┘
                ↓
  ┌─────────────────────────────────────┐
  │ 2. 清屏                              │
  │    ┌─────────────────────┐          │
  │    │ int 0x10, ah=0x06   │          │
  │    │ 功能：上卷窗口清屏   │          │
  │    └─────────────────────┘          │
  │    参数：                            │
  │    - al=0 (清全部行)                │
  │    - bh=0x07 (黑底白字)             │
  │    - cx=0 (左上角)                  │
  │    - dx=0x184f (右下角 79,24)       │
  └─────────────┬───────────────────────┘
                ↓
  ┌─────────────────────────────────────┐
  │ 3. 获取光标位置                      │
  │    ┌─────────────────────┐          │
  │    │ int 0x10, ah=0x03   │          │
  │    │ 功能：读取光标位置   │          │
  │    └─────────────────────┘          │
  │    输入：bh=0 (页号)                │
  │    返回：dh=行号, dl=列号            │
  └─────────────┬───────────────────────┘
                ↓
  ┌─────────────────────────────────────┐
  │ 4. 打印字符串 "1 MBR"                │
  │    ┌─────────────────────┐          │
  │    │ int 0x10, ah=0x13   │          │
  │    │ 功能：显示字符串     │          │
  │    └─────────────────────┘          │
  │    参数：                            │
  │    - al=0x01 (光标跟随)             │
  │    - bh=0 (页号)                    │
  │    - bl=0x02 (绿字)                 │
  │    - cx=5 (字符串长度)              │
  │    - dh/dl (光标位置，从步骤3获取)   │
  │    - es:bp (字符串地址)             │
  └─────────────┬───────────────────────┘
                ↓
  ┌─────────────────────────────────────┐
  │ 5. 死循环                            │
  │    jmp $ (无限循环，程序挂起)        │
  └─────────────────────────────────────┘
