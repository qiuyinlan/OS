# 显存

在电脑里，**屏幕上的字其实不是直接写上去的**。  
而是电脑里有一块**内存专门用来存显示的内容**，  
就像一张“黑板”，每一个格子都能放一个字。


在电脑启动后，显示器通常处于 **文本模式（text mode）**。  
这个模式下，屏幕被分成 25 行 × 80 列的字符格，每个格子能显示一个字母或符号。  
而显卡把这些字符存在内存的一个固定位置上——地址从 `0xB8000` 开始。

==一个字符占两个字节——第一个是字符，第二个是颜色。==

这就是显示用的地方（叫做“显存”）。  
如果你往这块地方写入 `'A'`，  
屏幕上就会显示一个 **A**！✨

```
0xB8000  = 屏幕第 1 行第 1 列
0xB8002  = 屏幕第 1 行第 2 列
...
0xB8FA0  = 屏幕第 25 行第 80 列

```

VGA = **Video Graphics Array**（视频图形阵列）
VGA 显示标准规定的：

- 彩色文本模式显存从 **物理地址 0xB8000** 开始；
    
- 黑白模式则是 `0xB0000`。

## VGA 有两种主要模式

|模式|用途|例子|每个格子内容|
|---|---|---|---|
|文本模式（Text Mode）|显示字符|“Hello OS!”|字符 + 颜色|
|图形模式（Graphics Mode）|显示图片、窗口、鼠标|游戏、桌面|每个像素点的颜色|

书里（《操作系统真象还原》）讲的是 **文本模式下的 VGA**，  
因为在写内核时，我们只想先能“显示字”，  
还不需要画图那么复杂的模式。
